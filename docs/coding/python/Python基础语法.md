# Python基础语法

缩进：Python使用缩进来确定代码之间的逻辑关系和层级关系，缩进可以使用制表符Tab或者空格，4个空格表示一个缩进（<font color="red">注意：Python3中不允许混用</font>）。代码缩进的量不会导致代码语义的改变，Python要求同一行代码必须有相同缩进的量，程序中如果代码缩进不同，可能程序输出的意思也不会痛，甚至产生错误。

换行：Python代码语句之间使用换行符相隔，但是在一行中可以用分号分隔多个语句，但是不推荐这么做。Python官方建议每行代码不超过79个字符，超过的部分可以用反斜杠`\`来换行。如果是字符串，还可以使用双引号每行单独括起来即可，并不影响字符串内容。

注释：单行注释使用`#`号标识，多行注释使用三引号`"""`或单引号`'''`括起来。

## 0、输入输出函数

输出（打印）函数`print()`：与其他打印函数区别不大，多个打印对象之间用逗号隔开即可，可以在最后一个参数使用具名参数`sep`指定分隔符（默认是空格），`end`指定每个对象末尾字符（默认是换行符）。

输入函数`input()`：`prompt`参数是可选的，用于显示提示信息，函数会等待用户输入内容并按下回车键，最后返回输入的字符串。
```python
name = input("请输入您的姓名：")
print(f"您好，{name}！")
```

## 1、变量与标识符

定义变量语法：`变量名/标识符 = 变量值/字面量`
  - 变量只有在赋值后才会创建
  - 首次使用变量会在内存中划分空间，再次使用不再划分，如果变更则修改原空间（变量）的值

标识符：由数字、字母、下划线组成，不能以数字开头。
  - 严格区分大小写
  - 不能使用关键字（保留字）作为标识符，详见Python官网：https://docs.python.org/3/reference/lexical_analysis.html#keywords
  - Python3中可以使用中文标识符，但不推荐
  - 用括号`()`包裹的表达式作为标识符，可以作为函数名，可以直接方位该标识符获取返回值，效果跟定义变量一样

变量命名规范
  - 变量和函数名：使用小写字母，单词间用下划线连接（[蛇形命名法](http://localhost:5173/coding/common/编程中常见的命名法.html#蛇形)）。
  - 常量：使用全大写字母，单词间用下划线连接。
  - 类名：使用大驼峰名命名规范，即每个单词首字母大写。
  - 前缀约定：单下划线开头表示内部使用标识（弱私有）；双下划线开头则为强私有；单下划线结尾则一般是为了与关键字冲突。此外，还要尽量避免与内部函数/类名冲突的变量名。

## 2、数据类型

数据类型
  - 数值类型
    - 整型：`int`
    - 浮点型：`float`
    - 布尔型：`bool`，只有`True`和`False`两个值，它可以当做整型（1是0否）。
    - 复数：`complex`，固定写法`z = a + bj`，其中`a`为复数的实部，`b`为复数的虚部，虚数单位`j`是固定的。
  - 字符串类型：`str`
  - 列表类型：`list`
  - 元组类型：`tuple`
  - 字典类型：`dict`
  - 集合类型：`set`
  - 字节类型：`bytes`
  - 其他类型：`None`

检测数据类型的函数：
  - `type()`：返回变量的类型
  - `isinstance()`：判断变量是否为指定类型

## 3、格式化输出

占位符：占据指定格式的字符串位置，可以在字符串中插入`%`、使用`format()`方法和格式化f等方式进行占位。

### 3.1 `%`的格式化输出

  | 符号 |                 描述                 |
  | :--: | :----------------------------------: |
  | `%c` |        格式化字符及其ASCII码         |
  | `%s` |             格式化字符串             |
  | `%d` |              格式化整数              |
  | `%u` |           格式化无符号整数           |
  | `%o` |         格式化无符号八进制数         |
  | `%x` |        格式化无符号十六进制数        |
  | `%X` |    格式化无符号十六进制数（大写）    |
  | `%f` | 格式化浮点数字，可指定小数点后的精度 |
  | `%e` |       用科学计数法格式化浮点数       |
  | `%E` | 作用同`%e`，用科学计数法格式化浮点数 |
  | `%g` |           `%f`和`%e`的简写           |
  | `%G` |           `%F`和`%E`的简写           |
  | `%p` |      用十六进制格式化变量的地址      |

```python
#!/usr/bin/env python3

# 基础示例，多个占位符占位值在%号后使用括号括起来
str_demo = "喜欢唱、跳、rap、篮球"
learning_time = 2.5
print("全民制作人们大家好，我是练习时长%.1f年的个人练习生xxx，%s" % (learning_time, str_demo))

# 补位
print("%5d" % 123) # 右对齐，前面不足5位补空格
print("%-5d" % 123) # 左对齐，前面不足5位补空格
print("%05d" % 123) # 右对齐，前面不足5位补0
print("%2d" % 123) # 占位值位数超过，占位结果为整个占位值

# 浮点数
print("%f" % 3.1415) # 浮点数占位，默认精度为6位小数，超出则四舍五入
print("%.2f" % 3.1415) # 指定精度为2位小数

# 百分号的占位符
print("%% -> %%" % ()) # 输出百分号%
```

## 4、运算符

### 4.1 算数运算符

加减乘除：`+-*/`，除法运算结果为浮点数

`//`：取整除（向下取整/截取）

`%`：取余（模）

`**`：幂运算，$m**n = m^n$

运算规则：
  - 先乘除，后加减
  - 同级运算从左往右
  - 可以用括号调整计算的优先级

**优先级**：幂 > 乘、除、取余、取整除 > 加减
