# 计算机中的数据存储

计算机中的数据可以分为三种类型：文本、图片和声音（视频为图集和声音的复合形式），而这些数据都是以二进制的形式来存储的。

## 1、常见的进制

* 二进制：由0和1组成，在编程语言字面量中一般以`0b`开头
* 十进制：由0~9组成
* 八进制：由0~7组成，在编程语言字面量中一般以`0`开头
* 十六进制：由0\~9和A\~F组成，A\~F表示10\~15的值，在编程语言字面量中一般以`0x`开头

> 注意：以上二进制、八进制、十六进制字面量开头写法是jdk7才加入的特性。
>
> 计算机使用二进制由来：最初的计算机程序编写后是用纸带打孔来进行存储的，其中纸带只有两种状态（打孔和未打孔），基于当时的硬件水平，就只能用二进制来表示这两种状态了，延续到后面使用范围较广的5伏电路板中，就规定电路中的电压小于3.3v为0，大于3.3v定为1。

## 2、进制间转换

* 任意进制转十进制：`系数 * 基数的权次幂 然后全部相加`，例如二进制的$1407_{八进制} = 7\times8^0 + 0\times8^1 + 4\times8^2 + 1\times8^3 = 775$

  * 系数：每一位上的数
  * 基数：当前进制数
  * 权：（小数点前）从右往左依次为0、1、2、3、4、5……

  > 可以了解一下二进制转十进制可以有一个“8421快速转换法”

* 十进制转其他进制：除基取余法，不断的除以基数得到余数，直到商为0，再将余数倒着拼起来即可。

## 3、文本数据

* 数字：`99 => 0110 0011`

* 字母：`'a' => 97（ASCII码） => 0110 0001`

* 汉字

  ① GB2312编码：1981年5月1日发布的简体中文汉字编码国家标准。收录7445个图形字符，其中包括6763个汉字。

  ② BIG5编码：台湾地区繁体中文标准字符集，共收录13053个中文字，1984年实施。

  ③ GBK编码：2000年3月17日发布，收录21003个汉字，包含国家标准GB13000-1中全部中日韩汉字和BIG5编码中的所有汉字。

  ④ Unicode编码：国际标准字符集，它将世界各种语言的每个字符定义一个唯一的编码，以满足跨语言、跨平台的文本信息交换。

> 说明
> ASCII码，即American Standard Code for Information Interchange，美国信息交换标准码表，是一种基于拉丁字母的字符编码方案，使用7位二进制数来表示128个字符，包括英文字母、数字、标点符号和一些控制字符。[ASCII码表](https://www.asciitable.com/)

## 4、图片数据

图片数据在计算机中分为三种，黑白图、灰度图和彩色图。

* 黑白图中，用0表示黑、1表示白，这样就可以用一个矩阵显示出不同的黑白图案。
* 灰度图则是在黑白图的基础上，每个点0表示纯黑，255表示纯白，然后0~255之间的每个数字表示不同程度的灰色，这样表示出来的矩阵则比黑白图多了灰度变化。
* 彩色图则又更进一步，利用光学三原色（红绿蓝，RGB）发出不同程度的光的原来来显示所有的颜色，红绿蓝分别使用0~255的值来表示其占比，如此即可显示出彩色图片，相比于灰度图又多出了色彩变化。

图片的存储还需要先了解显示器中的三个知识点：

* 分辨率：标识显示器包含的像素数量，例如常见的1920\*1080表示显示器横向每排又1920个像素，纵向每列又1080个像素。

  一般情况下，人们喜欢用分辨率中较大的那一个数值接近的整值来简述分辨率，例如1920\*1080，较大值接近2000，所以俗称为2K分辨率；2560\*1440为2.5K分辨率（但是现如今的情况是这个分辨率几乎顶替了1920\*1080了，一般都叫它2K）。

* 像素：显示器显示色彩的最小单元，是一种能根据存储的颜色参数值显示颜色的硬件，现在显示器的像素点一般包含了RGB三种颜色的发光源，每个光源根据RGB值显示不同大小的显示强度和亮度。

* 三原色：显示器中的三原色主要指的是光学三原色——红绿蓝（RGB），注意与原料三原色的红黄蓝区分。用这三种颜色即可显示出不同的颜色。

有了以上前置知识即可说清楚颜色如何存储在计算机中，即只需将每个像素点的RGB值获取到后再根据不同的图片格式（JPEG、PNG等）进行编码存储即可。

## 5、声音数据

声音是一种连续的模拟信号，在现代计算机中，要对声音进行存储需要对模拟信号的值在时序上进行采样（采样率很大程度上决定了存储的声音有多还原），如此就得到了一个时间和采样信号的关系数据，将这些数据编码保存即可。
