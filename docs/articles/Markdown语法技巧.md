# Markdown语法技巧

## 1、图片

图片语法：`[alt内容](img_url)`

也可以直接用img标签：`<img src="img_url" alt="提示内容"/>`

### 图片左对齐问题

一种方式是直接给Markdown的图片语法使用加粗语法，但是当图片需要缩放时，就需要使用img标签：`<img src="img_url" style="zoom:图片缩放百分比;" align="left"/>`，align也可以直接在style中使用float: left。

问题描述：当使用zoom进行缩放后，渲染结果中，文字环绕会与图片同行。

解决：在图片后面添加一个标签`<div style="clear: both;"></div>`，即可将文字换行，避免环绕。

## 2、锚点跳转

参考来源：[GitHub Markdown Tutorial](https://github.com/shengcaishizhan/GitHub_Markdown_Tutorial)以及AI

在文章浏览过程中，可以通过在文章中任意位置定义锚点实现超链接跳转到文章任意位置，锚点功能本质上是使用HTML标签中的ID来实现的。

### 2.1 定义锚点

#### 使用标题作为锚点

Markdown标题会自动称为锚点，其ID转换通常遵循以下规则：

* 空格变为`-`
* 小写字母
* 移除标点符号

例：`# 第一章 介绍 -> #第一章-介绍`

#### 手动定义锚点

直接在Markdown中使用HTML标签定义锚点：

```html
<!-- 可以使用span标签 -->
<span id="自定义锚点ID">锚点位置</span>

<!-- 也可以使用a标签 -->
<a id="自定义锚点ID">锚点位置</a>
```

> 注意：在锚点ID中避免使用 `#`, `%`, `&`等特殊字符

### 2.2 创建跳转链接

链接到统一文档内的标题（多用于创建目录）

```markdown
[跳转第一章](#第一章-介绍)
[超链接文本](#锚点ID)
```

链接到手动定义的锚点

```markdown
[跳转到锚点位置](#自定义锚点ID)
```

链接到其他文档的特定位置

```markdown
[查看另一文档的说明](文档路径#锚点ID)
```

### 2.3 特殊技巧

```markdown
[返回顶部](#)

[带标题的锚点链接](#锚点ID "标题")
```

完整示例：

```markdown
# 文档标题

## 目录
- [安装指南](#安装指南)
- [配置说明](#配置说明)
- [常见问题](#常见问题)

## 安装指南
安装步骤说明...

<a id="配置说明"></a>
## 配置说明
配置参数介绍...

## 常见问题
### 问题一
解答...

[返回目录](#目录) | [返回顶部](#文档标题)
```
